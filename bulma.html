NPM

É uma ferramenta que permite aos programadores de JavaScript instalar, gerenciar e compatilhar pacotes de forma rápida e acessível.


Funcionamentos dos comandos

Criando um projeto React: npx create-react-app meu-projeto-react cd meu-projeto-react npm start


Criando um projeto Angular: npm install -g @angular/cli ng new meu-projeto-angular cd meu-projeto-angular ng serve


Criando um projeto Next.js: npx create-next-app@latest meu-projeto-next cd meu-projeto-next npm run dev


Criando um projeto Vue.js: npm create vue@latest cd meu-projeto-vue npm install npm run dev
